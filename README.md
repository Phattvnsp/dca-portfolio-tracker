# dca-portfolio-tracker
Making an DCA portfolio tracker, by integrated previous data from excel to store with SQLite, and also learning other languages such as JS React ElysiaJS/Nextjs and so on.

# Project Overview :D
# 📈 DCA Portfolio Tracker

> Transitioning from static Excel sheets to a dynamic Full-Stack DCA Dashboard.
> Minimal, modern, pleasing to the eye style

## 🚀 Key Features
- **Smart Dashboard:** Summarize Total Invested, Current Value, and Total Gain/Lost.
- **Visual Insights:** Showing charts of investment distribution by individual stock and portfolio growth with `recharts`.
- **Simplified Logging:** Add Record system that supports complete information (Buy/Sell, Fee, Notes).
- **Transaction History:** Check historical records in detail and easy to read.

## 🎨 Design Identity
Design by keeping it **"Modern & Pleasing to the eye"**
- **Theme:** Modern Light Mode
- **Color Palette:** 
  - `Background:` #F8F6F0 (Off-white)
  - `Primary Accent:` #52F2D0 (Mint Green)
- **UX Layout:** Sidebar Navigation for easy access to Dashboard, Add Record, and History.

## 🛠 Tech Stack
- **Frontend:** 
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Icons:** Lucide React

## 📂 Project Structure
- `/dashboard` - Summary cards and performance graphs.
- `/add-record` - Transaction input form.
- `/history` - Comprehensive buy/sell logs.
